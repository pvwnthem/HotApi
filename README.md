

#### Base Url
```
  /api/v1/
```
#### Get Car by id
```
GET /api/v1/car/{id}
```

####  Example

```
curl -s -X GET /api/v1/car/
```

#### Get Recent Posts
```http
GET /api/v1/posts/recent
```
| Query     | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `?limit=` | `number` | **Not Required**. How many of the recent posts to grab |
